CodeSafe Pro

Advanced Password Manager

CodeSafe Pro is an advanced password manager. You only need one password to rule them all. CodeSafe Pro is fully compatible with KeePass – the famous open source password manager.

Download

CodeSafe Pro can be downloaded and installed from Windows Phone Store.

Key features

  • Create, edit, delete, and organize logins and notes
  • Browse or search for logins with ease
  • Generate cryptographically strong random passwords
  • Audit your password strength
  • Open/update/synchronize KeePass database files from multiple sources
  • Totally free with no ads

Supported sources

  • File association — open file from an email attachment, a website via Internet Explorer, an app, etc.
  • SD card
  • Dropbox
  • OneDrive/SkyDrive
  • WebDAV
  • Web server

Open source

CodeSafe Pro is based on work of Tran Ngoc Van. The original work can be found at 7Pass, of which Codesafe Pro is based off version 3.6.

Codesafe Pro is open source. The source code can be found on Github.
