CodeSafe Pro

CodeSafe Pro is an advanced password manager. You only need one password to rule them all. CodeSafe Pro is fully compatible with KeePass -- the famous open source password manager.

Key Features

  • Create, edit, delete, and organize logins and notes
  • Browse or search for logins with ease
  • Generate cryptographically strong random passwords
  • Audit your password strength
  • Open/update/synchronize KeePass database files from multiple sources **
  • Totally free with no ads

Supported Sources **

  • SD card
  • Dropbox
  • OneDrive/SkyDrive
  • WebDAV
  • Web server
